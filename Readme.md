# Student Performance Analytics Dashboard

## Overview

This project presents an interactive Power BI dashboard designed to analyze student academic performance, grade distribution, engagement, and pass rates across different departments.

The dashboard provides insights into student performance trends using various visualizations, KPI metrics, and interactive filters.

---

## Objectives

* Analyze student academic performance
* Compare grade distributions
* Monitor pass and fail rates
* Identify performance trends across departments
* Build an interactive analytical dashboard using Power BI

---

## Tools Used

* Power BI
* Data Visualization
* Dashboard Design
* Data Analysis

---

## Dashboard Features

* KPI Cards
* Interactive Slicers and Filters
* Department-wise Analysis
* Grade Distribution Analysis
* Pass/Fail Categorization
* Parent Education Analysis
* Comparative Performance Charts
* Interactive Visualizations

---

## Key Insights

* Grade B students showed the highest pass count.
* Student performance varied across departments.
* Parent education level influenced student performance trends.
* Overall pass rate exceeded 58%.
